# Server Home Model
description

## Installation
Requires BCM185 installation on the raspberry pi before hand

then:
Install dependencies:
```
npm install
```
To run the server
```
sudo node index.js
````

## Built with
- [Node.js](https://nodejs.org/en/)
- [Wiring-pi-node](https://github.com/WiringPi/WiringPi-Node)

## Dependencies
```
"node-dht-sensor": "0.0.34",
"node-fetch": "^2.1.2",
"wiring-pi": "^2.2.1"
```
